```
{product}-{role}-{subject}-{background}-{width}x{height}.{ext}
```

- product：
	- 製品・シリーズ名
- role：
	- 画像の用途・役割
- subject：
	- 画像の中身
- background：
	- 背景色
	- 透過：trans
- width：
	- サイズ：横幅
- height：
	- サイズ：縦幅
- ext：
	- 拡張子


### 実例

```
wood-repair-thumb-front-transparent-300x300.png
wood-repair-main-floor-white-1200x1200.jpg
```
