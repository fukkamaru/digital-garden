# ① ファストトラベル拠点として使う（＝実質ワープ装置）

ピンクの家は「自宅設定」ができるのが本質。

- 「メタモンのはた」を置く  
    → 自宅に設定  
    → ワープポイント化

### 有効活用の考え方

- マップの端・高台・作業拠点に置く
- 「よく行く場所＝1拠点」ではなく  
    → **移動のハブとして配置する**

### 具体例

- 資源採取エリアの近くに1軒
- 建築エリアの近くに1軒
- メイン拠点とは別に“作業拠点ハウス”

👉 **結果：移動時間が劇的に短縮**

# ② 「すみごこちブースター」として使う

👉 すみごこちが上がりやすい（＝住環境が優秀）
