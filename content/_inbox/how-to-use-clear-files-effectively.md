美術館や博物館で購入したクリアファイルの使い分けを考えてみる。というのも、わっちはあまりグッズ購入というものをしない。それでも、お金を出しても良いかなと思ったときは、なるべく実用性のあるものを購入する。

その１つがクリアファイルである。

デジタルが発達した今でも、紙の資料は少なくないので
用途に応じて分類わけするにはちょうど良いアイテムである

## 所有しているクリアファイルの確認

- mocha展：四季眺望セット（4種）
- ファンタジーアート展：天野嘉孝「F6：コシ・ファン・トゥッテ」
- ファイナルファンタジー：クリアファイルセット（4種）
- 日本国宝展：中村佑介コラボ
- 京都国立博物館：トラりん
- パウル・クレー展：蛾の踊り
- 大ゴッホ展：夜のカフェテラス
- 大原美術館：ポール・シニャック「オーウェルシーの運河」

## 役割を考える

- 一次集約（INBOX）
	- 一番のお気に入りを使う（使用頻度：高）
	- 今やっていることを全部ここ
	- 公私混同関係無し
- 仕事・公共系 （責任が発生する）
	- 見た目が落ち着いたもの、中身がひと目で分かる
	- 私情を入れない
	- 仕事、公共、「失くすと面倒」な紙
- 私用（趣味）
	- 持ち運び用
	- 家限定
- 長期保存・参照用（ARCHIVE）
	- 入れたら基本出さない
	- アーカイブとして利用


## 持ち運び

最大3枚に収める
- 常用
- 責任・確定用
- 私用（持ち運び用）

> - 言語化できるものだけ入れっぱなしにする
> - 見えないコストの適正化（重さ、探索時間、判断）

### ローテーション

私用（趣味）については、家の中だけで使うものと、外出先に持ち出すので分ける。
このとき、外出先に持ち出すクリアファイルについては、ローテーションで使い分けられるようにすることで
気分をコントロールする。

例えば、
春→夏→秋→冬で絵柄が変わると、  
リュックから取り出した瞬間に

> 「あ、いま◯月だな」

と無意識に把握できる。

判断を楽にするために追加でルールを設ける
- 今の季節の1枚だけ入れっぱなし
- 季節が終わったら無条件で入れ替え
- 中身は「今季触る個人資料だけ」

これだけで、
- 使ってない紙が溜まらない
- 自然に棚卸しが起きる
###  おすすめ基準（実用向け・月固定）

| 季節   | 月            |
| ---- | ------------ |
| 🌸 春 | **3・4・5月**   |
| 🌻 夏 | **6・7・8月**   |
| 🍁 秋 | **9・10・11月** |
| ❄️ 冬 | **12・1・2月**  |
