switch2でゲーム配信ではないが、自分のプレイを録画・録音する場合、機材は何が必要？

  

Switch2で**「配信はしないが、自分のプレイを録画・録音したい」**場合、必要な機材は目的によって2パターンあります。

---

# ① 一番シンプル（本体機能だけ）

### 必要機材

- **Nintendo Switch 2本体**
    
- **microSD Expressカード**
    

### 方法

Switch2の  
**キャプチャーボタン → 動画保存**

### 特徴

- 最大 **直近30秒程度**（Switch1と同じ仕様なら）
    
- 音声は**ゲーム音のみ**
    
- 自分の声は録れない
    
- 画質はゲーム解像度そのまま
    

### 向いている用途

- プレイ記録
    
- SNS投稿
    
- バグ記録

YouTube投稿や検証動画などならこちら。

### 必要機材

4. **録画ソフト**

録画時間に対するファイルサイズはどれくらいになりますか？

  

Switch2の録画サイズは、**解像度・フレームレート・ビットレート**で決まります。  
キャプチャーボード＋OBSで録画するケースを基準に、現実的な目安を示します。

---

# 録画サイズの目安（一般的設定）

|画質|ビットレート|1時間の容量|
|---|---|---|
|720p 30fps|5 Mbps|約2.2 GB|
|1080p 30fps|8 Mbps|約3.6 GB|
|1080p 60fps|12 Mbps|約5.4 GB|
|1080p 60fps 高画質|20 Mbps|約9 GB|

---

# 計算の仕組み（参考）

容量は基本的に

```
ビットレート × 録画時間
```

で決まります。

例  
1080p60fps / 12Mbps

```
12 Mbps × 3600秒 = 43200 Mb= 約5.4GB
```

（録画ではなく**リアルタイム配信**）

---

# 1 YouTube側の準備

まず **ライブ配信機能を有効化**します。

手順

```
YouTube Studio↓作成↓ライブ配信を開始
```

初回のみ

```
電話番号認証
```

が必要です。

※有効化まで **最大24時間**かかる場合があります。

---

# 2 配信キーを取得

YouTube Studio

```
ライブ配信↓ストリーム↓ストリームキー
```

ここで表示される

```
ストリームキー
```

をコピーします。

---

# 3 OBS設定

使用ソフト

OBS

```
設定↓配信
```

設定

|項目|設定|
|---|---|
|サービス|YouTube|
|サーバー|Primary|
|ストリームキー|貼り付け|

---

# 4 映像設定

```
設定↓映像
```

|項目|設定|
|---|---|
|基本解像度|1920×1080|
|出力解像度|1920×1080|
|FPS|60|

---

# 5 配信ビットレート

```
設定↓出力↓出力モード：詳細↓配信
```

|項目|設定|
|---|---|
|エンコーダ|x264|
|レート制御|CBR|
|ビットレート|6000 kbps|
|キーフレーム|2|

※YouTube推奨

---

# 6 配信開始

OBS

```
配信開始
```

を押す。

その後

YouTube Studioで

```
ライブ開始
```

を押します。

---

# 配信の流れ

```
Switch2↓LGX2↓OBS↓YouTube↓ライブ配信
```

配信終了後

**自動で動画が保存されます。**

---

# 容量について

ライブ配信の場合

|保存場所|容量|
|---|---|
|PC|基本0GB|
|YouTube|保存される|

つまり

**4〜6時間プレイしてもPC容量は使いません。**

あなたが最初に聞いたことは

答えは **その通りです。**

### 方法A：録画 → アップロード

→ **PC容量は基本使わない**

|ボタン|意味|
|---|---|

**配信だけ押せばローカル保存はされません。**

- **4K HDMI出力**
    
- **DisplayPort Alt Mode対応**
    
- **映像出力対応**
    

と書いてあるものを選びます。

# 2. HDMI入力対応（キャプチャーボード用）

```
iPad↓USB-C → HDMI↓LGX2↓PC
```

なので

- **HDMI出力アダプタ**
    
- **HDMIオス**
    

タイプならOKです。

# 3. 充電ポート付きがおすすめ

```
USB-C├ HDMI└ USB-C PD（充電）
```

こういう形。

理由  
HDMI出力中は **バッテリーが減り続ける**ため。

---

# 4. 60Hz対応

安い製品は30Hzのものがあります。

最低でも

```
1080p 60Hz
```

対応。

---

# 5. 有名メーカーを選ぶ

安物は

- 発熱
    
- 映像途切れ
    
- 相性問題
    

が起きます。

安全なメーカー

- Anker
    
- UGREEN
    
- Belkin
    
- Apple純正
