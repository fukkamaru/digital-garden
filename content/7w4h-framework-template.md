| **タイプ** | **単語**    | 問い（簡易） | **日本語の問い？**      | **意味**         |
| ------- | --------- | ------ | ---------------- | -------------- |
| W       | who       | 誰が？    | 誰が責任者なのか？        | 主・責任者          |
|         | with whom | 誰と？    | 誰と協働するのか？        | 対象・共同者         |
|         | to whom   | 誰に？    | 誰に届けるのか？/納品するのか？ | 対象・顧客          |
|         | witch     | どの？    | どの手段・製品・チャネルなのか？ | 採用する製品・チャネル・手段 |
|         | what      | 何を？    | 何を行うのか？          | 施策・成果物         |
|         | when      | いつ？    | いつ行うのか？          | 時期・スケジュール      |
|         | whare     | どこで？   | どこで行うか？          | 場所・媒体・市場       |
|         | why       | なぜ？    | なぜ行うか？           | 目的・理由          |
| H       | how       | どうやって？ | どのように行うか？        | 手段・ワークフロー      |
|         | how many  | どれくらい？ | どれだけの数量？         | 数量・件数・規模       |
|         | how long  | どれくらい？ | どれだけの期間？         | 期間・工数・時間       |
|         | how much  | いくら？   | いくらかかるか？         | 費用・予算          |
